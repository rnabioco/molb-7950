Advanced summarizing with across()

Apply functions to multiple columns at once.

library(palmerpenguins)
penguins |>
  group_by(species) |>
  summarise(

      list(mean = ~ mean(.x, na.rm = TRUE), sd = ~ sd(.x, na.rm = TRUE))

Multiple grouping variables

Complex grouping for detailed analysis.

penguins |>
  group_by(species, island, year) |>
  summarise(
    n_penguins = n(),
    avg_mass = mean(body_mass_g, na.rm = TRUE),
    mass_range = max(body_mass_g, na.rm = TRUE) -
      min(body_mass_g, na.rm = TRUE),

We have 4 legends - can they be condensed?

Yes, but it is not exactly straightforward.

# fetch the legend for `p1`
legend <- get_legend(
  p + theme(legend.position = "bottom")
)

p <- p + theme(legend.position = "none")

# first `plot_grid` builds the panels
panels <- plot_grid(
  p,
  p,
  p,
  p,
  labels = c(
    "A",
    "B",
    "C",
    "D"
  ),
  nrow = 2
)

# second `plot_grid` adds the legend to the panels
plot_grid(
  panels,
  legend,
  ncol = 1,
  rel_heights = c(1, .1)
)
